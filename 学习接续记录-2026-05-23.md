## 四、踩坑速记（累计）

1. **pnpm 11** → `pnpm approve-builds esbuild vue-demi core-js`
2. **登录失败** → SQL 无用户，需注册；或 `admin` + `.env` 的 `ADMIN_PASSWORD`
3. **发文要管理员** → `UPDATE blog_user SET role=1 WHERE username='yulorde3122';`
4. **封面一直转圈** → `blog-server\src\upload\local` 存在 + 重启后端
5. **8888 浏览器空白感** → 正常，只有一行「欢迎 这是后台server首页」
6. **F12 找接口** → 切 **Network** 标签（不是 Elements）→ 勾选 Preserve log → F5 → 过滤 `getArticleById`
7. **Clear 按钮** → Network 工具栏 🚫 图标；列表空时可直接 F5，不必 Clear
8. **温馨提示弹窗** → 接口 `code: 100`；常因 `blog_config` 未初始化或未保存
