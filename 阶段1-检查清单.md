## G. 今日已解决的问题（2026-05-23）

| 现象 | 原因 | 处理 |
|------|------|------|
| 登录「用户名不存在」 | SQL 无初始用户 | 注册 `yulorde3122` 或 INSERT test 用户 |
| `ERR_PNPM_IGNORED_BUILDS` | pnpm 11 拦截构建脚本 | `pnpm approve-builds esbuild vue-demi core-js` |
| 后台新建文章页打不开 | 缺 `@element-plus/icons-vue` | `pnpm add @element-plus/icons-vue` |
| `vue-demi package.json` 报错 | CDN 插件启动时加载 | 改 `getCdnPlugin()` 懒加载 |
| 封面一直「图片上传中」 | 无 `src/upload/local` 目录 | 已创建目录；`UPLOADTYPE=local` |
| 前台点文章一直加载 | 详情 API 失败 + 无 catch | 已修 `article.vue` / `article.js` / 后端 `getArticleById` |
| 后台分类字典异常 | API 路径 `Category` 大小写错 | 已改 `api/category.ts` |

## H. 本阶段你要会的 5 句话

1. **blog-server** 提供接口，**blog-v3** 给读者看，**blog-v3-admin** 给你写文章。  
2. **MySQL** 存数据，**.env** 告诉后端怎么连 MySQL。  
3. 开发时要开 **3 个终端**，先后端再两个前端。  
4. 前台/后台的 `pnpm run serve` 是开发模式；上线时才用 `build`。  
5. 阶段 1 成功 = 本地能逛、能发文；**还不需要买服务器**。
